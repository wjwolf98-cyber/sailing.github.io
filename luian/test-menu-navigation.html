测试说明

这个测试页面用于验证 prototype/pc-system.html 的菜单导航功能是否正常工作。

  1. 打开 prototype/pc-system.html
  2. 点击左侧菜单项（如"用户管理"）展开子菜单
  3. 点击子菜单项（如"用户列表"）查看对应页面

结论

✓ 所有菜单项和页面ID都正确匹配！

如果点击菜单无法显示页面，可能的原因：

  • 需要先点击父菜单项（如"用户管理"）展开子菜单
  • 浏览器控制台可能有JavaScript错误
  • CSS样式可能导致页面被隐藏

使用步骤：

  1. 点击父菜单：先点击"用户管理"、"角色管理"等父菜单项，展开子菜单
  2. 点击子菜单：再点击子菜单项（如"用户列表"），即可显示对应页面
  3. 直接菜单："工作台"和"数据统计"可以直接点击

快速测试

在浏览器中打开 prototype/pc-system.html，然后按F12打开开发者工具，在控制台中运行以下命令测试：

// 测试显示用户列表页面
showPage('brand-list');

// 测试显示视频课程页面
showPage('training-video');

// 测试显示学习进度页面
showPage('training-progress');
